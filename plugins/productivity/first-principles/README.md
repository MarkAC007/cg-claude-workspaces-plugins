# First Principles Plugin

Physics-based reasoning framework for deconstructing problems to fundamental truths, challenging assumptions, and reconstructing optimal solutions. Based on Elon Musk's first principles methodology — avoid reasoning by analogy, rebuild from irreducible facts.

## Commands

| Command | Description |
|---------|-------------|
| `/first-principles:deconstruct` | Break any problem into constituent parts and irreducible truths |
| `/first-principles:challenge` | Systematically classify constraints as hard/soft/assumption |
| `/first-principles:reconstruct` | Build an optimal solution using only hard constraints |

## Workflow

```
Deconstruct → Challenge → Reconstruct
   (Parts)    (Classify)   (Build new)
```

## Prerequisites

No external dependencies.

## Install

```bash
claude plugin add ~/GitHub/cg-claude-workspaces-plugins/plugins/productivity/first-principles
```
