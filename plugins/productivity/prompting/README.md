# Prompting Plugin

Meta-prompting system for generating and optimising AI prompts using Anthropic's Claude 4.x best practices. Produce well-structured, production-ready prompts that consistently deliver high-quality outputs.

## Commands

| Command | Description |
|---------|-------------|
| `/prompting:generate` | Generate a structured prompt from a plain-language description |
| `/prompting:optimize` | Audit and improve an existing prompt against best practices |

## Prerequisites

No external dependencies. Works with any Claude Code workspace.

## Install

```bash
claude plugin add ~/GitHub/cg-claude-workspaces-plugins/plugins/productivity/prompting
```
