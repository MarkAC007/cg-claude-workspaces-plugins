# Evals Plugin — Setup

## Prerequisites

- Claude Code CLI

## API Keys

None required for command guidance workflows.

## Install

```bash
claude plugin add ~/GitHub/cg-claude-workspaces-plugins/plugins/dev-tools/evals
```

## Usage

```
/evals:create-use-case
/evals:create-judge
/evals:run-eval
/evals:compare-prompts
/evals:compare-models
/evals:view-results
```
